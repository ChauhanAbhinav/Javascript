JavaScript Prompt

JavaScript provides three methods for interacting with users: alert, prompt, and confirm.
The alert method displays a message to the user and waits for them to click "OK".
The prompt method displays a message and an input field, allowing the user to enter a value.
The confirm method displays a message and two buttons, "OK" and "Cancel", allowing the user to make a choice.

let name = prompt("What is your name?", "Guest");
alert("Hello, " + name + "!");
let isBoss = confirm("Are you the boss?");
alert( name + " is " + (isBoss ? "" : "not ") + "the boss"); // true if OK is pressed
